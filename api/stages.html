PATCHapi/stages/:stageId

Update a Stage

This endpoint allows you to update a stage. Add the properties to be updated in the request body.

Request Headers

  • Content-Type: application/json

Request Body

{
  "status": "IN_PROGRESS"
}

Response Body

{}
